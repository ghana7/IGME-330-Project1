Bouncing Letters

By Jimmie Harkins





Watch the letters bounce into words!

If a letter ball hits the center, it will start a word. Then other letters can join it to keep building a word. If a ball bounces off another, that means no word can be made. Once a word is finished, the last letter will change color and make a word of its own. Words can branch, too! If a ball hits the middle of a word and can make a new word, it'll change color and start its own branch.

Controls:

  • Add Ball: adds a new ball, placed randomly
  • Reset: resets the simulation
  • Scatter Balls: releases all the balls from the center, disabling collision for a few seconds
  • Ball Speed: controls how fast new balls move
  • Ball Size: controls how big new balls are
  • Automatically Repopulate Balls: sets whether a new ball is created whenever one is added to a word
